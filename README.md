<a name='Function'></a>
| 功能名 | 介绍 |
|---|---|
| HEX偏移 | 两种模式，手动输入和选择文件。**手动输入**：输入符合正则表达式[0-9A-F]+的内容，选择偏移量后点击左或右偏移(-或+)，即刻在底部输出框内看见内容；**文件选择**: 选择文件后，在选择偏移量后点击左或右偏移，即可对文件进行覆盖偏移，在此之前会自动备份文件。 |
| SQLite数据库编辑器 | SQL数据库是常用的数据库文件，本软件**使用安卓提供的原生SQL数据库接口**，对选择的数据库文件进行分析，并将其输出为Excel表格，进行可视化编辑、读取。支持使用SQL命令，支持搜索等功能。**修改后你需要手动保存** |
| 自动点击 | **需要*悬浮窗*权限和*无障碍*权限**，悬浮窗权限用于在屏幕上显示可拖动和点击的按钮，无障碍权限用模拟点击。在授予所需权限后，你可以新建配置，并在*配置-管理点击处*添加点击坐标(x, y坐标，，**必填，最多支持三位小数**)、延迟(*可选，**单位ms***)、备注(*可选*)。您可以在*开发者设置*中，找到并开启*指针位置*，开启后，在屏幕上方会显示你当前触摸点的坐标 |
